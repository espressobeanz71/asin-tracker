About ROI Calculation

True Cost = Product Cost + Prep Fee + (Weight x Shipping Rate)
Amazon Fees = (Sale Price x Referral %) + FBA Fee
Net Profit = Sale Price - Amazon Fees - True Cost
ROI = Net Profit / True Cost x 100
